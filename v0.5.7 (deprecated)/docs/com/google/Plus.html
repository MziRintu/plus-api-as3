authorize()method
public static function authorize(callback:Function, view:StageWebView, clientId:String, redirectUri:String, scopes:Array):void

Every request that your application sends to the Google+ API needs to identify your application to Google. We use an OAuth 2.0 access token. You should use an access token when you are making calls on behalf of a given user.

Parameters

callback:Function — Handle method for the response.
 
view:StageWebView — StageWebView object for handle the Google+ Login.
 
clientId:String — Client ID from your Google Project on Developer Console.
 
redirectUri:String — Redirect URL from your Google API Project on Developer Console. Ex. http://www.example.com
 
scopes:Array — Scopes are Array strings that enable access to particular resources, such as user data.
