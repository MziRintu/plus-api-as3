public static function list(callback:Function, userId:String, collection:String, maxResults:int, pageToken:String = null):void

List all of the activities in the specified collection for a particular user.

Parameters

callback:Function — (Required) Handle Method for list activities
 
userId:String — (Required) The ID of the user to get activities for. The special value "me" can be used to indicate the authenticated user.
 
collection:String — (Required) The collection of activities to list. Acceptable values are: Activities.COLLECTION_PUBLIC
 
maxResults:int (default = NaN) — (Optional) The maximum number of activities to include in the response, which is used for paging. For any response, the actual number returned might be less than the specified maxResults. Acceptable values are 1 to 100, inclusive. (Default: 20)
 
pageToken:String (default = null) — (Optional) The continuation token, which is used to page through large result sets. To get the next page of results, set this parameter to the value of "nextPageToken" from the previous response.

search()method 
public static function search(callback:Function, query:String, language:String = null, orderBy:String = null, maxResults:int, pageToken:String = null):void

Search public activities.

Parameters

callback:Function — (Required) Handle method for Activities search
 
query:String — (Required) Full-text search query string.
 
language:String (default = null) — (Required) Specify the preferred language to search with. See https://developers.google.com/+/api/search#available-languages for available values.
 
orderBy:String (default = null) — (Optional) Specifies how to order search results. Acceptable values are: Activities.ORDER_BY_BEST or Activities.ORDER_BY_RECENT
 
maxResults:int (default = NaN) — (Optional) The maximum number of activities to include in the response, which is used for paging. For any response, the actual number returned might be less than the specified maxResults. Acceptable values are 1 to 20, inclusive. (Default: 10)
 
pageToken:String (default = null) — (Optional) The continuation token, which is used to page through large result sets. To get the next page of results, set this parameter to the value of "nextPageToken" from the previous response. This token can be of any length.
