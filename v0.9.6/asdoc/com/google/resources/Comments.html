public static function list(callback:Function, activityId:String, maxResults:int, sortOrder:String = null, pageToken:String = null):void

List all of the comments for an activity.

Parameters

callback:Function — (Required) Handle Method for list comments for an activity
 
activityId:String — (Required) The ID of the activity to get comments for.
 
maxResults:int (default = NaN) — (Optional) The maximum number of comments to include in the response, which is used for paging. For any response, the actual number returned might be less than the specified maxResults. Acceptable values are 0 to 500, inclusive. (Default: 20)
 
sortOrder:String (default = null) — (Optional) The continuation token, which is used to page through large result sets. To get the next page of results, set this parameter to the value of "nextPageToken" from the previous response.
 
pageToken:String (default = null) — (Optional) The order in which to sort the list of comments. Acceptable values are: Comments.ASC or Comments.DESC
