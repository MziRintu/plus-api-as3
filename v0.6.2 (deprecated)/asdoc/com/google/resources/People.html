get()method
public static function get(callback:Function, userId:String):void

Get a person's profile. If your app uses scope https://www.googleapis.com/auth/plus.login, this method is guaranteed to return ageRange and language.

Parameters

callback:Function — (Required) Handle Method for get People Information
 
userId:String — (Required) The ID of the person to get the profile for. The special value "me" can be used to indicate the authenticated user.

list()method 
public static function list(callback:Function, userId:String, collection:String, maxResults:int, orderBy:String = null, pageToken:String = null):void

List all of the people in the specified collection.

Parameters

callback:Function — (Required) Handle method for list people.
 
userId:String — (Required) Get the collection of people for the person identified. Use "me" to indicate the authenticated user.
 
collection:String — (Required) The collection of people to list. Acceptable values are: People.COLLECTION_CONNECTED or People.COLLECTION_VISIBLE
 
maxResults:int (default = NaN) — (Optional) The maximum number of people to include in the response, which is used for paging. For any response, the actual number returned might be less than the specified maxResults. Acceptable values are 1 to 100, inclusive. (Default: 100)
 
orderBy:String (default = null) — (Optional) The order to return people in. Acceptable values are: People.ORDER_BY_ALPHABETICAL or People.ORDER_BY_BEST
 
pageToken:String (default = null) — (Optional) The continuation token, which is used to page through large result sets. To get the next page of results, set this parameter to the value of "nextPageToken" from the previous response.

search()method 
public static function search(callback:Function, query:String, language:String = null, maxResults:int, pageToken:String = null):void

Search all public profiles.

Parameters

callback:Function — (Required) Handle method for Search profiles.
 
query:String — (Required) Specify a query string for full text search of public text in all profiles.
 
language:String (default = null) — (Optional) Specify the preferred language to search with. See https://developers.google.com/+/api/search#available-languages for available values.
 
maxResults:int (default = NaN) — (Optional) The maximum number of people to include in the response, which is used for paging. For any response, the actual number returned might be less than the specified maxResults. Acceptable values are 1 to 50, inclusive. (Default: 25)
 
pageToken:String (default = null) — (Optional) The continuation token, which is used to page through large result sets. To get the next page of results, set this parameter to the value of "nextPageToken" from the previous response. This token can be of any length.
